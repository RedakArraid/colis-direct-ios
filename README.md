# ColisDirect — iOS Application

Ce répertoire est destiné à accueillir la version iOS de l'application ColisDirect.

## Structure recommandée
- Utilisation de Swift / SwiftUI pour une intégration native optimale ou intégration d'une WebView hybride reprenant les optimisations mobiles de la version Android.
- Charte graphique : Orange (#FF6C00), Blanc (#FFFFFF), et Noir (#0F0F0F).
- Utilisation des mêmes structures de données et API que les versions Web et Android.
